## Criterios de calificación
Evalúo cada solución con una nota de 0 a 10 considerando:
1. **Corrección** — ¿produce el output correcto para todos los casos?
2. **Estilo Python** — ¿sigue las convenciones de Python (PEP 8)?
3. **Simplicidad** — ¿la solución es apropiadamente simple para el nivel del problema?
4. **Robustez** — ¿maneja edge cases razonables?
